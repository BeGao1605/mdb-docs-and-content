- API

Import

Via JavaScript:

import { MdbPopoverModule } from 'mdb-angular-ui-kit/popover'; â€¦ @NgModule ({ ... imports: [MdbPopoverModule], ... })

Inputs

Name Type Default Description
animation Boolean true Apply a fade transition to the popover
mdbPopover String '' Default content value if mdbPopover input isn't present.
mdbPopoverTitle String '' Default title value if mdbPopoverTitle input isn't present.
delayShow Number 0 Delay showing the popover (ms)
delayHide Number 0 Delay hiding the popover (ms)
placement String 'top' How to position the popover - top | bottom | left | right
trigger String 'click' How popover is triggered - click | hover | focus | manual. You may pass multiple triggers; separate them with a space. manual cannot be combined with any other trigger.
offset Number 0 Offset of the popover relative to its target

Outputs

Name Description
popoverShow Fires when show animation starts.
popoverShown Fires when show animation ends.
popoverHide Fires when hide animation starts.
popoverHidden Fires when hide animation ends.

Methods

Name Description Example
hide Manually closes a popover popover.hide()
show Manually opens a popover popover.show()
toggle Manually toggle a popover popover.toggle()
@ViewChild('popover') popover: MdbPopoverDirective; ngAfterViewInit(): void { this.popover.show(); }
