Show with trigger

You can manually show alert using triggerRef property with other element.

Hidden alert!
import React, { useRef } from 'react'; import { MDBAlert, MDBContainer, MDBBtn } from 'mdb-react-ui-kit'; export default function App() { const triggerRef = useRef(null); return ( Show alert! Hidden alert! ); }
